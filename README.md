# Cybersecurity Studies

Structured, practical cybersecurity learning repository focused on AppSec, CloudSec, and ML Security.

This repo is designed for consistent weekly study with real outputs:
- Lab write-ups
- Portfolio projects
- Interview-ready notes
- Career and certification planning (India-focused)

## Who This Repo Is For

- Beginners starting cybersecurity from a software/ML background
- Learners with limited weekly time (about 6 hours/week)
- Job switchers targeting security analyst, AppSec, cloud security, or ML security roles

## How To Start Using The Repo

1. Open [prep helpers/STUDY_INDEX.md](prep helpers/STUDY_INDEX.md).
2. Complete [prep helpers/BEGINNER_START_HERE.md](prep helpers/BEGINNER_START_HERE.md) to set up your lab and baseline concepts.
3. Read [prep helpers/syllabus.md](prep helpers/syllabus.md) for the full learning map.
4. Execute week by week using [prep helpers/execution_plan_12_ml_app_cloud.md](prep helpers/execution_plan_12_ml_app_cloud.md).
5. Use [prep helpers/HANDS_ON_EXERCISES.md](prep helpers/HANDS_ON_EXERCISES.md) and document every finding.
6. Build projects from [prep helpers/PORTFOLIO_PROJECTS.md](prep helpers/PORTFOLIO_PROJECTS.md).
7. Prepare applications with [prep helpers/INTERVIEW_AND_NETWORKING.md](prep helpers/INTERVIEW_AND_NETWORKING.md) and [prep helpers/TEMPLATES_PACK.md](prep helpers/TEMPLATES_PACK.md).

## Recommended Weekly Flow (6 Hours)

- 2 hours: Core study module reading
- 2 hours: Hands-on labs
- 1 hour: Notes and finding write-up
- 1 hour: Portfolio or interview preparation

## What You Should Have By Week 12

- 12 to 18 lab write-ups
- 3 portfolio-ready security projects
- Threat models and remediation notes
- Interview stories tied to real artifacts
